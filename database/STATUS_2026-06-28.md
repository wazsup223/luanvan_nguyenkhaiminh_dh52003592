## 5. 🟢 Thanh Toán — Hướng Dẫn Demo

### Cách 1: Simulation Mode (Không cần tunnel) ⭐ CHO DEMO
Dùng `POST /api/payment/confirm-simulation` để mô phỏng thanh toán thành công:

```bash
# 1. Tạo đơn → nhận token
POST /api/payment/momo/create
→ Trả về: { payUrl: "#", token: "SIM-MOMO-xxxx" }

# 2. Confirm simulation (mô phỏng thanh toán thành công)
POST /api/payment/confirm-simulation
Body: { transactionToken: "SIM-MOMO-xxxx", amount: 50000, method: "momo" }
→ DB: transaction = success, order.payment_status = paid ✅
```

→ **Ưu điểm:** Không cần tunnel, không cần thẻ test, nhanh
→ **Phù hợp:** Demo offline, thuyết trình

### Cách 2: ZaloPay Sandbox Real (Cần tunnel)
1. Chạy `cloudflared tunnel --url http://localhost:3001`
2. Copy URL tunnel (vd: `https://xxx.lhr.life`)
3. Cập nhật `.env`:
   ```
   ZALOPAY_CALLBACK_URL=https://xxx.lhr.life/api/payment/zalopay/callback
   ```
4. Restart backend
5. Demo:
   - Tạo đơn → nhận QR code (dùng ZaloPay app thật scan)
   - Dùng test card từ docs.zalopay.vn:
     - **Thẻ thường:** `9704456000000006` / `10/25` / `123456`
   - ZaloPay server gọi callback → DB cập nhật tự động

→ **Phù hỉnh:** Demo online, test thực tế payment flow
